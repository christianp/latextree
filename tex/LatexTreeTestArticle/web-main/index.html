Unlisted macros and environments

With LatexDocument.make
website(), the contents of unlisted environments can be hidden by wrapping the environment in a div of class unlisted, which has the property display:none set in the stylesheets. There is a problem however with macros having one or more arguments that are not listed in 'macrosdef.py'. Its absence causes pylatexenc.latexwalker to not parse the argument(s) but instead to parse the macro as having zero arguments, and to parse the argument as a separate LatexGroupNode.
