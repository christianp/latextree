7.2.3  The video environment

The video environment
Here is a floating video environment which uses the custom includevideo command. This command is intended to mirror the way that includegraphics is used within a floating figure environment. In the PDF version this is again rendered as a hyperlink, but LatexTree embeds the video into the webpage. To get an embed code for a YouTube video, click on Share underneath the video then Embed and it will generate an entire iframe tag. You can recover the url from this quite easily.
